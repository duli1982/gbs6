🚀 AI-Powered Features with Gemini for Workspace

Application Hidden/Powerful AI Feature How to Use/What it Does
Sheets Help me analyze (Gemini Analyst) An "on-demand analyst" that finds trends, points out outliers, suggests next steps for data deep-dives, and quickly generates clear, interactive charts from your data via natural language prompts. (Coming soon/New)
Sheets Generate Formulas & Actions Ask Gemini in the side panel to generate complex formulas, apply conditional formatting, create pivot tables, or build dropdown menus. It can perform multi-step edits from a simple prompt.
Sheets AI Function A capability to run multiple text-based actions at scale from a cell formula, such as generating, summarizing, or categorizing text across many rows based on specific instructions.
Docs Help me refine More than just a text generator, this feature acts as a writing coach, offering thoughtful suggestions to strengthen arguments, improve structure, and clarify key points. (New)

Sheets/Docs/Slides Reference Drive Files (Context) When prompting Gemini, you can reference other files in your Drive (e.g., "Summarize the main points from @Meeting Notes Doc") to generate context-aware content.

📈 Sheets: Data Visualization and Analysis
Feature Description How to Find/Use
Sparklines Tiny charts within a single cell that quickly visualize trends. Use the =SPARKLINE() function.
Heatmaps Uses color scales to highlight high/low values, outliers, or errors. Format > Conditional formatting > Color Scale.
Filter Views Create, name, and save specific filter/sort settings that won't affect collaborators. Data > Filter views > Create new filter view.
QUERY Function Acts like an SQL-lite database language inside your spreadsheet for complex data handling. =QUERY(data_range, "select * where...")
IMPORTRANGE Function Connects and imports a range of cells from another Google Sheet. =IMPORTRANGE("url", "SheetName!A1:C10")
Smart Chips for Data Automatically populate a cell with data like a person's name, file, or date. Type the @ symbol in a cell.

📝 Docs & Slides: Collaborative and Structural Enhancements
Feature Description How to Find/Use
Pageless View Removes page boundaries, allowing you to write on an "infinitely long" digital plane. File > Page setup > Pageless tab.
In-document Checklists Create interactive checklists that you can mark as complete directly in your document. Format > Bullets & Numbering > Checklist, or type [] + space.
Insert Chart from Sheets (Linked) Embed a chart from a Sheet into a Doc/Slide and update it with one click. Insert > Chart > From Sheets.
Custom Shortcuts (Docs) Define your own keyboard shortcuts for special characters or custom scripts. Tools > Preferences > Substitutions.
Bookmarking and Internal Links (Docs) Create anchors within a long document to build a clickable internal structure. Insert > Bookmark, then Insert > Link.
Live Meetings in Document Join a Google Meet call directly from within the Sheets, Docs, or Slides file. (Premium) Top navigation bar.
