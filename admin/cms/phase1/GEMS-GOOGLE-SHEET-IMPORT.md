## Import Instructions

### Method 1: Direct JSON Import (Recommended)

1. **Open the JSON file:**
   - Location: `admin/cms/phase1/GEMS-MIGRATION-DATA.json`
   - This file contains all structured data ready for import

2. **Import to Google Sheets:**
   - Open your existing Gems Google Sheet
   - Go to File → Import
   - Select "Upload" tab
   - Choose the JSON file
   - Select "Insert new sheet(s)" or "Append to current sheet" depending on your preference

3. **Format the data:**
   - Ensure columns align with your existing structure
   - Verify all special characters (emojis) display correctly
   - Check that long text fields (prompts, descriptions) are properly formatted

### Method 2: Manual Copy-Paste from Tables Below

Use the comprehensive data tables below to manually copy and paste gem information into your Google Sheet.
